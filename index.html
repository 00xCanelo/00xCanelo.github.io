                        About 00xCanelo
                        ]
                    

                    

                        
Senior Offensive Security Researcher with expertise in web application exploitation, binary
                            analysis, and source code auditing. Specialized in discovering subtle logic flaws, type
                            confusion vulnerabilities, and novel attack primitives through deep architectural analysis.
                        

                        
Active CTF competitor with top rankings in major competitions including DEFCON, Google CTF,
                            and HackTheBox. Passionate about knowledge sharing through detailed technical writeups that
                            explore the "why" behind exploitation techniques, not just the "how".

                        
Contributing to the security community through open-source tool development and maintaining
                            vulnerability research frameworks. When not breaking things, I'm building tools to help
                            others break things better.
